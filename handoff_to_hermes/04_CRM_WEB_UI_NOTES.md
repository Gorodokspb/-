# 04 — CRM web UI notes

## Общий стиль
Нужен плотный, практичный интерфейс для ежедневной работы.

## Уже сделано по web estimate editor
- Упрощена шапка сметы.
- Убраны лишние описательные блоки.
- Убран `Project ID` из рабочего экрана.
- Большой нижний блок заменён на компактный submit strip.
- Quick-add переработан в компактный inline composer.
- Добавлены действия для базы расценок, шаблонов, операций и калькулятора.
- Правая панель показывает счётчик расценок.
- Убрано ограничение показа расценок до 60 элементов.

## Projects dashboard UX pass — 2026-04-24
По скриншоту пользователя со страницы `Объекты и сметы`:
- Убран большой маркетинговый баннер `Все ключевое по проектам...` как лишняя информация.
- Убран блок `Быстрые действия` из левого sidebar — действия перенесены наверх.
- Убран дублирующий блок `Выбранный объект` под метриками; рабочая таблица теперь начинается выше.
- В верхнюю панель добавлены/выделены кнопки: `+ Проект`, `+ Контрагент`, `Открыть смету`, `Карточка проекта`, `Финансы`, `Прайс-лист`.
- Добавлен regression test `tests/test_projects_dashboard_template.py` на отсутствие лишних блоков и наличие верхних действий.

Дополнительный UX-pass по следующему скриншоту:
- Убран индикатор `Синхронизация OK` из sidebar как лишняя информация.
- Список `Мои объекты` в sidebar теперь выводит все проекты, кроме статуса `Завершен`, без ограничения `projects[:6]`.
- Верхние основные действия зафиксированы в одну строку по центру; `Сменить пароль` вынесен в пользовательский блок справа, чтобы не разбивать ряд рабочих кнопок.
- Regression test расширен проверками по sidebar и topbar layout.
