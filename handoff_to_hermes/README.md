# Handoff to Hermes — DekorCRM

Дата обновления: 2026-04-24 19:40 UTC

Эта папка — очищенная рабочая память проекта для Hermes. Здесь фиксируются только проектные решения, текущий статус, безопасные инструкции и следующие шаги.

## Правило безопасности
Не сохранять сюда:
- пароли;
- GitHub/Telegram/API токены;
- реальные значения `.env`;
- приватные ключи;
- персональные данные клиентов сверх необходимого рабочего контекста;
- полные сырые диалоги, если в них есть секреты.

Вместо секретов писать только, где и каким способом они настраиваются, без значений.

## Индекс
- `00_PROJECT_CONTEXT.md` — что это за проект и где он лежит.
- `01_CURRENT_STATUS.md` — текущее состояние ветки, коммитов и последних работ.
- `02_DECISIONS.md` — принятые решения и предпочтения пользователя.
- `03_GITHUB_AND_DEPLOY.md` — GitHub, ветки, push/deploy, без токенов.
- `04_CRM_WEB_UI_NOTES.md` — заметки по веб-CRM.
- `05_ESTIMATE_EDITOR_NOTES.md` — редактор смет, UX и проверки.
- `06_SECRETS_AND_SECURITY.md` — правила обращения с секретами.
- `07_NEXT_STEPS.md` — ближайшие задачи.
- `CHANGELOG.md` — история обновлений этой handoff-папки.
